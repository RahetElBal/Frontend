# BeautiQ - دليل العميل (العربية)

هذا الدليل يوضح وظيفة كل صفحة حسب الدور.

## لوحة الإدارة (Superadmin)

### لوحة تحكم الإدارة
- نظرة عامة على المنصة (الصالونات، المستخدمون، الإيرادات، حالة النظام).
- عرض الصالونات والمستخدمين الجدد.

### المستخدمون
- إنشاء وإدارة المستخدمين.
- تعيين الأدوار والوصول للصالونات.
- البحث والتصفية.

### الصالونات
- إنشاء صالون وتعيين المالك.
- عرض تفاصيل وحالة الصالون.
- إدارة معلومات الاتصال.

### الخدمات (للسوبرأدمن فقط)
- اختيار صالون لعرض خدماته.
- البحث والتصفية حسب الفئة.
- تجميع الخدمات حسب الفئة.
- تعديل الأسعار لكل خدمة (لكل صالون).
- حفظ جميع التعديلات أو إعادة الضبط.

### إعدادات الإدارة
- إدارة إعدادات المنصة (إن وُجدت).

## لوحة الصالون (Admin / User)

### لوحة التحكم
- إحصائيات يومية (الإيرادات، المواعيد، العملاء).
- ملخصات سريعة والخدمات الأكثر طلباً.

### جدول المواعيد (Agenda)
- إنشاء موعد (عميل، خدمة، تاريخ، وقت).
- تعديل الموعد وإضافة ملاحظات.
- إلغاء الموعد فقط إذا لم يتم الدفع.
- وضع الموعد كمكتمل.
- تسجيل الدفع بعد إكمال الموعد.
- الإشعارات:
  - تذكير بالمواعيد القادمة.
  - تنبيه المواعيد غير المدفوعة والمتأخرة.

### العملاء
- إضافة وتعديل بيانات العملاء.
- البحث بالاسم أو البريد أو الهاتف.
- عرض تاريخ العميل.

### الخدمات (عرض فقط)
- استعراض الخدمات حسب الفئة.
- عرض التفاصيل (السعر، المدة، الوصف).

### المبيعات
- عرض المدفوعات وتفاصيل المبيعات.
- إنشاء بيع يدوي عند الحاجة.
- المدفوعات المرتبطة بالمواعيد تُسجل تلقائياً.

### التحليلات
- متابعة الإيرادات والمواعيد ومتوسط الفاتورة.
- التصفية حسب الفترة (يومي/أسبوعي/شهري/سنوي).
- عرض أكثر الخدمات طلباً.

### إعدادات الصالون
- إعداد معلومات الصالون.
- ضبط قواعد الحجز وساعات العمل.
- إعداد الإشعارات والإيصالات.

### الإعدادات
- إدارة الملف الشخصي وتفضيلات الحساب.
