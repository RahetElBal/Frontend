# BeautiQ - Client Guide (English)

This guide describes what each page does for all roles.

## Admin Panel (Superadmin)

### Admin Dashboard
- View platform overview (salons, users, revenue, system status).
- See recent salons and users.

### Users
- Create and manage users.
- Assign roles and salon access.
- Search and filter users.

### Salons
- Create salons and assign an owner.
- View salon details and status.
- Manage salon contact info.

### Services (Superadmin Only)
- Select a salon to view its services.
- Search and filter services by category.
- Group services by category.
- Edit prices per service (per salon).
- Save all changes or reset edits.

### Admin Settings
- Manage platform-level settings (if enabled).

## Salon Panel (Admin / User)

### Dashboard
- View daily stats (revenue, appointments, clients).
- See top services and quick summaries.

### Agenda
- Create appointments by selecting client, service, date, and time.
- Edit appointments and add notes.
- Cancel appointments only if unpaid.
- Mark appointments as completed.
- Record payment for completed appointments.
- Notifications:
  - Upcoming appointment reminders.
  - Overdue unpaid appointment alerts.

### Clients
- Add and update client profiles.
- Search clients by name, email, or phone.
- View client history.

### Services (View)
- Browse service catalog by category.
- View service details (price, duration, description).

### Sales
- View all payments and sale details.
- Create sales manually when needed.
- Payments linked to appointments are recorded automatically.

### Analytics
- Track revenue, appointments, and average ticket.
- Filter by period (daily/weekly/monthly/yearly).
- View top services.

### Salon Settings
- Configure salon details and business settings.
- Adjust booking rules and working hours.
- Configure notifications and receipts.

### Settings
- Manage your user profile and account preferences.
